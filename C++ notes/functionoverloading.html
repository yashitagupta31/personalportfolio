FUNCTION OVERLOADING


What is Function Overloading?

Function overloading is a feature of C++ where more than one functions can have the same name but have different parameters. This is a form of polymorphism.

The function name is taken same and the arguements are taken different in the function being overloaded. Function can be overloaded in two ways:


  • using different data types

  • using different arguements

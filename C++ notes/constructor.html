CONSTRUCTOR


What is Constructor?

Constructor is a user defined function used for initializing of variables.

There is no use of any keyword such as void, int, float, char etc. while declaration of constructor.
The name of constructor is same as that of class.
There is no need to call a constructor unlike function.

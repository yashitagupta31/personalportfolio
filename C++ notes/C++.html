CLASS

Class in C++ is the building block that leads to Object-Oriented programming. A class is a blueprint for the object.

We can think of a class as a sketch of a house.It contains all the details about the floors, doors, windows, etc. The sketch is the class.


OBJECT

Object is an instance of a class that has the properties and behaviours of the class attatched

In the above example, based on the descriptions we build the house. House is the object.


DATA MEMBERS

The variables used in a class are referred as Data Members.


MEMBER FUNCTION

The functions of class are referred as Member Function.


MEMBER ACCESSIFIERS

Data members or member functions in a class have to defined as public, private or protected which defines that who can access this members.

Private

Data type and member functions under private can be accessed only in that class.

Public

Data type and member functions under public can be accessed anywhere in the program.

Protected

Data type and member functions under protected can be accessed only by friend class.


ENCAPSULATION

It is the process of binding an object with data member or member function is called encapsulation.

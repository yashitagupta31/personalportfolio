OPERATOR OVERLOADING


What is Operator Overloading?

Operator overloading allows us to make operators work for user-defined classes. Unlike normal functions, the name of an operator function is always the operator keyword followed by the symbol of the operator, and operator functions are called when the corresponding operator is used.

FILE HANDLING


What is File handling?

File handling includes all the operations related to file assessment.

The modes in file handling can be:

  • in (used to read a file)

  • out (used to write in a file)


There are some in-built functions related to file operations which are stored in a header file- <fstream.h>.These operations are:

  • f.open (for opening a file)
  • f.close (for closing a file)

FILE HANDLING



What is file handling?

File handling in C is the process in which we create, open, read, write, and close operations on a file.
For this operations the program code needs a new header file named #include<stdlib.h>.
To carry out these operations one must ensure that the file being used in operations is saved in the BIN folder.

There are 3 modes of file operation in C:

  • Read ("r")
  • Write ("w")
  • Append ("a")


There are some in-built functions for differen file operation. These are:
  • fopen() for opening a file.
  • fclose() for closing a file.
  • fprintf() and fputs() for printing text in a file.
  • fscanf() and fgets() for reading text in a file.

UNION



What is Union?

Union is a user-defined data type that contains different variables of different data types in the same memory location. Unlike structure only one variable can be assigned a value in union.




Syntax Input Output 
#include<stdio.h>
void main()  
{
union abc
{           
int a,b,c;
char d,e,f;
float g,h,i;      
}
union abc p;          
p.a=10;
_____;
_____;
getch();
}

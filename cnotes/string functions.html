STRING FUNCTIONS



The following are the string functions in C:-

  • strlen:- To determine the length of the string.

  • strlwr:- To get the string written in lower case.

  • strupr:- To get the string written in upper case.

  • strcpy:- To copy the string from one location to another.

  • strcat:- To combine two strings and the result is returned to the first string.

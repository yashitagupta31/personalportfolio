STRUCTURE



What is Structure?

Structure in C is a user-defined data type used to bind two or more similar or different data types into a single type. It makes the program code organised and easy to read.
Struct keyword is used to declare a structure. Structure can be declared in or before main.
Value to the variables declared in structure can be given through object only.

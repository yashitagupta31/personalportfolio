POINTER



What is pointer?

  • Pointer is a variable which is used to store address of another variable.

  • A pointer variable is declared with a '* ' symbol. for eg: int *x;

  • While assigning a value to the pointer variable '& ' operator is used before another variable whose address is being given to the pointer. for eg: x=& a;

  • To reach to the value of the variable which is being pointed out '*' symbol is used. for eg: printf("%d",*x);

  • Pointer to Pointer Variable: This variable is used to point out a pointer variable. Here we use '**'. for eg: int**y;

DATA TYPES



What are Data Types?

Data type is a memory storing place. It is used to avoid wastage of memory and unnecessary use of memory. A data type stores a particular type of data.


Integer

Integer data type includes all the negative values, positive values and zero.
  • int is the keyword used for Integer data type.
  • %d is used to print/scan the integer data type.


Input Output 
#include<stdio.h>
main()
{
int a,b,c,d,e;
clrscr();
a=10;
b=20;
c=30;
d=40;
e=50;
printf("%d\n%d\n%d\n%d\n%d",a,b,c,d,e);
}


Float

Float data type includes all the integer values as well as decimal values.
  • float is the keyword used for Float data type.
  • %f is used to print/scan the Float data type.

Input Output 
#include<stdio.h>
main()
{
float a,b,c,d,e;
clrscr();
a=10.5;
b=20.6;
c=30.7;
d=40.8;
e=50.9;
printf("%f\n%f\n%f\n%f\n%f",a,b,c,d,e);
}
